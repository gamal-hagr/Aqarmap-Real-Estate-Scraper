# 🚀 Automated Real Estate Data Scraper (Aqarmap)

A professional Python-based web scraping tool designed to extract, clean, and structure real estate data from **Aqarmap Egypt**. This project demonstrates core Data Engineering skills such as data acquisition, handling pagination, and export automation.

## 🛠️ Tech Stack
- **Python 3.x**
- **BeautifulSoup4** (HTML Parsing)
- **Pandas** (Data Structuring & Export)
- **Requests** (HTTP Communication)

## ✨ Key Features
- **Stealth Mode:** Implements random delays and User-Agent rotation to mimic human behavior and avoid IP blocking.
- **Automated Pagination:** Automatically crawls through multiple search result pages (tested up to 16+ pages).
- **Comprehensive Data Capture:** Extracts Property Title, Price, and direct Property URL.
- **Structured Output:** Saves all extracted data into a clean, ready-to-analyze Excel file (`.xlsx`).
